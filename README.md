# Vibecraft Music Player

A premium, high-fidelity music experience powered by YouTube (unofficial), unofficial api of jiosavn and gaana.

## Getting Started

To get started, click the link above or https://music-player-wheat-gamma.vercel.app


## Features

- **Ad-Aware Playback**
- **High Fidelity UI:** Built with ShadCN, Tailwind CSS, and Framer Motion for a smooth, app-like feel.
- **Instant Search:** Debounced search across the YouTube Music category.
- **Audio & Video access**
- **Songs with different Audio Quality**
